# Web 站点技术方案

## 🎯 MVP 核心功能

### 1. 专业树形展示
- 三级分类结构：学科门类 → 专业类 → 具体专业
- 可展开/折叠的树形导航
- 快速搜索和筛选

### 2. 专业详情页
- 专业基本信息（代码、名称、描述、难度、学习时长）
- 完整技能清单
- 学习路径
- 相关专业推荐

### 3. 技能清单页
- 技能列表展示
- 技能详情弹窗
- 批量复制功能

### 4. Agent 学习功能
- 选择专业
- 一键复制技能清单
- 学习进度追踪（后续）

---

## 🏗 技术架构

### 技术栈
- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **部署**: Vercel
- **数据**: 静态 JSON (data/majors.json, data/skills.json)

### 项目结构
```
web/
├── app/
│   ├── layout.tsx          # 全局布局
│   ├── page.tsx            # 首页
│   ├── majors/
│   │   ├── page.tsx        # 专业列表页
│   │   └── [id]/
│   │       └── page.tsx    # 专业详情页
│   └── api/
│       └── majors/
│           └── route.ts    # 专业数据 API
├── components/
│   ├── MajorTree.tsx       # 专业树组件
│   ├── MajorCard.tsx       # 专业卡片组件
│   ├── SkillList.tsx       # 技能列表组件
│   └── CopyButton.tsx      # 复制按钮组件
└── data/
    ├── majors.json         # 专业数据（从根目录复制）
    └── skills.json         # 技能数据（从根目录复制）
```

## 🎨 页面设计

### 首页 (page.tsx)
- 项目介绍
- 专业分类概览
- 快速入口

### 专业列表页 (majors/page.tsx)
- 左侧：专业树形导航
- 右侧：专业卡片列表
- 顶部：搜索和筛选

### 专业详情页 (majors/[id]/page.tsx)
- 专业标题和基本信息
- 技能清单表格
- 学习路径时间线
- 复制技能清单按钮

---

## 📡 数据流

```
data/majors.json → app/api/majors/route.ts → 组件
                    ↓
                JSON API
                    ↓
              客户端组件渲染
```

---

## 🚀 部署方案

### Vercel 部署
1. 连接 GitHub 仓库
2. 配置根目录为 `web`
3. 自动部署（每次 push 自动触发）

### 环境变量
- 无需环境变量（纯静态数据）
