## 技能概述

机器学习基础是人工智能领域的核心技能，旨在让 AI Agent 掌握机器学习的基本概念、原理和常用算法。该技能使智能体能够理解数据驱动的决策机制，识别问题类型并选择合适的机器学习方法，为后续深入学习和实际应用奠定坚实基础。

## 核心能力

### 1. 机器学习概念理解
- **监督学习**: 通过标注数据训练模型进行预测（分类、回归）
- **无监督学习**: 发现未标注数据的内在模式和结构（聚类、降维）
- **强化学习**: 通过与环境交互获得奖励信号学习最优策略
- **半监督学习**: 结合少量标注数据和大量未标注数据进行学习

### 2. 常用算法掌握
- **线性模型**: 线性回归、逻辑回归、岭回归、Lasso回归
- **决策树**: 决策树算法、随机森林、梯度提升树（GBDT）
- **支持向量机**: SVM分类器、SVM回归
- **朴素贝叶斯**: 高斯朴素贝叶斯、多项式朴素贝叶斯
- **K近邻算法**: KNN分类、KNN回归

### 3. 模型评估能力
- **评估指标**: 准确率、精确率、召回率、F1-score、ROC-AUC、RMSE、MAE
- **交叉验证**: K折交叉验证、留一交叉验证
- **偏差-方差权衡**: 理解过拟合和欠拟合问题

### 4. 数据预处理
- **特征工程**: 特征选择、特征提取、特征构造
- **数据清洗**: 缺失值处理、异常值检测、数据标准化
- **数据转换**: 类别编码（Label Encoding、One-Hot Encoding）、数据归一化

## 学习路径

### 阶段一：概念入门（学习时长：30分钟）

3. 理解训练集、验证集、测试集的作用
4. 掌握过拟合和欠拟合的概念

2. 掌握决策树的基本原理

1. 使用sklearn库实现简单分类器
2. 完成数据预处理流程
3. 进行模型评估和优化
4. 解决实际分类或回归问题

## 常见问题

**Q1: 什么时候应该使用机器学习？**
A: 当问题具有明确的目标变量（分类或回归）、有足够的历史数据、且规则难以明确编码时，适合使用机器学习。

**Q2: 如何选择合适的机器学习算法？**
A: 首先考虑问题类型（分类/回归/聚类）、数据规模、数据特征类型，然后从简单算法开始尝试，根据评估结果逐步优化。

**Q3: 什么是特征工程？为什么重要？**
A: 特征工程是从原始数据中提取、构造和选择特征的过程。好的特征工程能显著提升模型性能，往往比选择复杂算法更有效。
