## 📋 专业概述

### 专业描述
培养具备利用数据分析工具和技术，从数据中提取有价值信息并支持决策能力的专业人才。涵盖数据采集、清洗、分析、可视化及报告生成等全流程能力。

### 对应岗位
- 数据分析师（Data Analyst）
- 业务分析师（Business Analyst）
- BI 工程师（BI Engineer）
- 数据运营专员（Data Operations Specialist）

- 中级：15K-30K/月

## 🎯 核心技能清单

### SK-IT-022：SQL 查询

**技能描述**：能够使用 SQL 语言从关系型数据库中高效查询、筛选、聚合数据。

**学习时长**：30-50 小时

**难度等级**：3/5

**核心知识点**：
1. 基础查询
   - SELECT、FROM、WHERE 子句
   - 排序（ORDER BY）
   - 限制结果集（LIMIT、OFFSET）

2. 条件过滤
   - 比较运算符（=, >, <, >=, <=, <>）
   - 逻辑运算符（AND, OR, NOT）
   - IN, BETWEEN, LIKE 操作符
   - NULL 值处理（IS NULL, IS NOT NULL）

3. 聚合函数
   - COUNT, SUM, AVG, MIN, MAX
   - GROUP BY 分组
   - HAVING 过滤分组

4. 多表连接
   - 内连接（INNER JOIN）
   - 左连接（LEFT JOIN）
   - 右连接（RIGHT JOIN）
   - 全连接（FULL JOIN）

5. 子查询与窗口函数
   - 子查询（标量子查询、行子查询、表子查询）
   - 窗口函数（ROW_NUMBER, RANK, DENSE_RANK）
   - 窗口函数（SUM, AVG, COUNT OVER）

**实践项目**：
- 查询电商订单数据，生成销售报表
- 分析用户行为数据，计算用户留存率
- 多维度统计分析（按时间、地区、品类）

**测试标准**：
- 能够编写复杂的 SQL 查询（涉及多表连接、子查询）
- 查询性能优化（索引使用、查询计划分析）
- 通过率 >= 80%

**参考资料**：
- W3Schools SQL 教程
- LeetCode SQL 题目（简单、中等难度）
- MySQL/PostgreSQL 官方文档

### SK-IT-023：Excel 高级应用

**技能描述**：掌握 Excel 高级功能，能够高效处理、分析、可视化数据。

**学习时长**：30-50 小时

**难度等级**：2/5

**核心知识点**：
1. 高级公式与函数
   - 查找函数（VLOOKUP, XLOOKUP, INDEX+MATCH）
   - 文本函数（LEFT, RIGHT, MID, TEXT, CONCAT）
   - 日期时间函数（TODAY, NOW, DATEDIF, EOMONTH）
   - 逻辑函数（IF, IFERROR, IFS）
   - 统计函数（SUMIF, SUMIFS, COUNTIF, COUNTIFS）

2. 数据透视表
   - 创建数据透视表
   - 字段拖拽与布局
   - 计算字段与计算项
   - 切片器与时间轴

3. 数据分析工具
   - 排序与筛选（高级筛选）
   - 数据验证（下拉列表、自定义规则）
   - 条件格式（数据条、色阶、图标集）
   - 数据分组与拆分

4. 图表制作
   - 柱状图、折线图、饼图、散点图
   - 组合图表（双轴图、瀑布图）
   - 动态图表（使用名称管理器、公式）
   - 图表美化与格式调整

5. 宏与 VBA（进阶）
   - 录制宏
   - 编辑 VBA 代码
   - 自定义函数

**实践项目**：
- 创建销售数据分析报表（数据透视表 + 图表）
- 制作动态财务预测模型
- 批量处理数据（数据验证 + 条件格式）

**测试标准**：
- 能够独立完成数据分析报表制作
- 使用数据透视表进行多维度分析
- 通过率 >= 85%

**参考资料**：
- Excel 官方帮助文档
- B站/YouTube Excel 教程
- Microsoft Learn Excel 课程

### SK-IT-024：BI 工具（Tableau/Power BI）

**技能描述**：能够使用 BI 工具创建交互式仪表盘，实现数据可视化与自助分析。

**学习时长**：40-60 小时

**难度等级**：3/5

**核心知识点**：
1. 数据连接与准备
   - 连接多种数据源（Excel、SQL、API）
   - 数据清洗与转换
   - 数据模型设计（星型模型、雪花模型）
   - 关系定义与管理

2. 数据可视化
   - 基础图表（柱状图、折线图、饼图、地图）
   - 高级图表（树状图、散点图、盒须图）
   - 仪表盘设计原则
   - 交互式控件（筛选器、参数）

3. 计算字段与度量值
   - DAX 公式（Power BI）
   - LOD 表达式（Tableau）
   - 时间智能函数（YTD、YoY、MoM）
   - 自定义计算字段

4. 仪表盘发布与分享
   - 发布到 Power BI Service / Tableau Online
   - 行级安全性（RLS）
   - 定时刷新数据源
   - 分享与协作

**实践项目**：
- 创建销售业绩仪表盘（多维度分析）
- 制作用户增长趋势看板
- 构建实时运营监控大屏

**测试标准**：
- 能够独立完成交互式仪表盘设计
- 数据模型设计合理
- 通过率 >= 80%

**参考资料**：
- Power BI 官方文档
- Tableau 官方文档
- BI 工具社区论坛

### SK-IT-025：报表生成

**技能描述**：能够根据业务需求，生成结构化、可读性强的数据分析报告。

**学习时长**：20-30 小时

**难度等级**：2/5

**核心知识点**：
1. 报告结构设计
   - 标题与摘要
   - 核心指标（KPI）
   - 数据展示（表格、图表）
   - 结论与建议

2. 数据可视化原则
   - 选择合适的图表类型
   - 色彩搭配与设计美学
   - 数据标注与说明
   - 图表标题与坐标轴标签

3. 报告撰写技巧
   - 数据解读方法
   - 趋势分析与归因
   - 异常值解释
   - 可行性建议

4. 自动化报表
   - 使用 Excel 模板
   - 使用 Python/Power Automate 自动生成
   - 定时发送报表邮件

**实践项目**：
- 生成周度销售数据报告
- 制作用户行为分析报告
- 输出季度业务复盘报告

**测试标准**：
- 报告结构清晰，数据准确
- 图表可视化效果良好
- 通过率 >= 85%

**参考资料**：
- 数据报告撰写指南
- 数据可视化原则（Edward Tufte）
- 业务分析案例

### SK-IT-026：趋势分析

**技能描述**：能够运用统计方法和分析工具，识别数据趋势，预测未来走向。

**学习时长**：30-50 小时

**难度等级**：3/5

**核心知识点**：
1. 基础统计分析
   - 描述性统计（均值、中位数、标准差）
   - 分布分析（正态分布、偏度、峰度）
   - 相关性分析（皮尔逊相关系数）
   - 假设检验（t检验、卡方检验）

2. 时间序列分析
   - 时间序列分解（趋势、季节性、随机性）
   - 移动平均法（SMA、EMA）
   - 趋势拟合（线性回归、多项式回归）
   - 季节性调整

3. 预测方法
   - 简单预测（均值预测、移动平均预测）
   - 趋势外推法
   - 指数平滑法
   - ARIMA 模型（进阶）

4. 业务指标分析
   - 同比分析（Year-over-Year）
   - 环比分析（Month-over-Month）
   - 复合增长率（CAGR）
   - 漏斗分析、留存分析

**实践项目**：
- 分析电商 GMV 趋势并预测下季度业绩
- 用户增长趋势分析与预测
- 产品销量季节性分析

**测试标准**：
- 能够正确使用统计方法分析趋势
- 预测结果具有合理性
- 通过率 >= 75%

**参考资料**：
- 统计学基础教程
- 时间序列分析课程
- Python 库（pandas, statsmodels）
