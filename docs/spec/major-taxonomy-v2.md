## 📋 文档说明

本文档基于人类高等教育专业体系和职业技能框架，重新设计 AI Agent 的专业体系和技能树结构。

**核心理念**：专业 = 完整的职业能力体系，技能 = 细分能力单元

---

## 🎯 核心概念重新定义

### 1.1 专业（Major）

**定义**：专业对应人类高等教育中的专业或职业岗位，是 Agent 能够独立承担某项工作的完整能力体系。

**特征**：
- ✅ 完整性：覆盖岗位所需的全套能力
- ✅ 独立性：无需其他专业即可独立工作
- ✅ 层次性：可分为基础专业、进阶专业
- ✅ 综合性：包含多个技能的组合

**示例**：
- ✅ **计算机专业**：完整覆盖软件开发、系统运维、数据分析等能力
- ✅ **会计专业**：完整覆盖财务核算、报表分析、税务处理等能力
- ❌ **邮件管理**：这只是单一技能，不是完整专业

### 1.2 技能（Skill）

**定义**：技能是专业内的细分能力单元，对应职业技能框架中的具体能力点。

**特征**：
- ✅ 原子性：不可再分的最小能力单元
- ✅ 可组合：多个技能组合形成专业
- ✅ 可复用：同一技能可被多个专业使用
- ✅ 可量化：可通过测试验证掌握程度

**示例**：
- ✅ **邮件分类技能**：自动分类邮件的能力
- ✅ **Python 编程技能**：使用 Python 编写代码的能力
- ✅ **数据清洗技能**：清洗和整理数据的能力

## 🌳 AI Agent 专业树形结构

### 2.1 三级分类体系

参考中国高等教育三级分类（学科门类 → 专业类 → 具体专业），设计 AI Agent 专业树：

```
学科门类（一级）
    └── 专业类（二级）
            └── 具体专业（三级）
                    └── 技能清单
```
