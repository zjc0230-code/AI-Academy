# AI 剧本创作技能描述（AC-1-3）

## 技能定义

AI 剧本创作是指利用人工智能技术进行影视、舞台、动画等剧本的创作，包括故事构思、场景设计、角色对话、情节编排、分镜设计等多个方面。该技能要求 AI 具备强烈的戏剧冲突构建能力、对话创作能力和视觉化叙事能力。

## 核心能力

### 1. 故事构思

**能力描述**：
能够构思完整、引人入胜的故事框架，包括主题立意、冲突设计、人物关系、情节走向等核心要素。

**关键要素**：
- **主题立意**：明确剧本的核心思想、价值观传递、情感共鸣点
- **冲突设计**：设计强烈的戏剧冲突（人与自我、他人、社会、自然的冲突）
- **人物关系**：构建复杂且有张力的人物关系网络
- **情节走向**：规划故事的起承转合，确保节奏合理
- **悬念设置**：在关键节点埋下悬念，推动观众持续关注

**执行步骤**：
1. 确定剧本类型（电影、电视剧、舞台剧、动画等）
2. 提炼核心主题和立意
3. 设计主要冲突和次要冲突
4. 构建人物关系图谱
5. 规划故事大纲和情节节点
6. 设置关键悬念和转折点
7. 确定故事节奏和高潮位置

### 2. 场景设计

**能力描述**：
能够设计具有画面感和氛围感的场景，包括环境设定、时空转换、镜头语言、视觉意象等。

**关键要素**：
- **环境设定**：场景的时间、地点、氛围、道具等细节
- **时空转换**：合理设计场景之间的切换逻辑
- **镜头语言**：运用推拉摇移、特写、全景等镜头技巧
- **视觉意象**：设计具有象征意义的视觉符号
- **氛围营造**：通过环境描写传递情感基调

**执行步骤**：
1. 确定场景的功能和目的
2. 设定场景的时间、地点、环境
3. 描述场景的视觉细节（光影、色彩、道具）
4. 设计人物在场景中的位置和动作
5. 运用镜头语言增强表现力
6. 通过环境描写营造氛围
7. 确保场景转换流畅自然

### 3. 角色对话

**能力描述**：
能够创作符合角色性格、推动情节发展、具有戏剧张力的对话，包括台词、潜台词、独白等。

**关键要素**：
- **角色声音**：每个角色都有独特的语言风格和表达方式
- **潜台词**：对话背后的真实意图和情感
- **冲突推动**：对话推动矛盾升级或解决
- **信息传递**：通过对话传递必要信息，避免说教
- **情感表达**：对话体现角色的情感变化和内心世界

**执行步骤**：
1. 分析角色性格和说话风格
2. 明确对话的目的和冲突
3. 设计对话的核心信息点
4. 运用潜台词增加层次感
5. 确保对话推动情节发展
6. 加入角色的情绪和态度
7. 检查对话的自然度和真实性

### 4. 情节编排

**能力描述**：
能够合理安排情节节奏，包括起承转合、高潮设计、伏笔回收、结局收尾等。

**关键要素**：
- **起承转合**：经典四段式结构，情节发展自然流畅
- **高潮设计**：精心设计情感和冲突的高峰时刻
- **伏笔回收**：前期伏笔得到合理呼应
- **节奏控制**：张弛有度，避免拖沓或仓促
- **结局收尾**：给观众满意的情感归宿

**执行步骤**：
1. 规划故事整体结构（起承转合）
2. 设计关键情节节点
3. 确定高潮时刻和表现形式
4. 设置伏笔并规划回收时机
5. 调整情节节奏和张力
6. 设计合理的结局
7. 检查情节逻辑和连贯性

### 5. 分镜设计

**能力描述**：
能够将文字剧本转化为可视化的分镜脚本，包括镜头设计、画面构图、动作指导、音效提示等。

**关键要素**：
- **镜头设计**：选择合适的镜头类型和运动方式
- **画面构图**：合理安排人物、背景、道具的位置关系
- **动作指导**：描述人物的具体动作和表情
- **音效提示**：标注背景音乐、音效、对话的配合
- **时长控制**：估算每个镜头的大致时长

**执行步骤**：
1. 分析场景内容和情感基调
2. 选择合适的镜头类型和角度
3. 设计画面构图和人物位置
4. 描述人物动作和表情细节
5. 标注音效和对话配合
6. 估算镜头时长
7. 绘制或文字描述分镜草图

## 适用场景

- **影视创作**：电影、电视剧、网剧、微电影等剧本创作
- **舞台创作**：话剧、音乐剧、舞台剧等剧本创作
- **动画创作**：动画片、漫画、游戏剧情等剧本创作
- **广告创作**：商业广告、品牌宣传片等剧本创作
- **短视频创作**：短视频系列、微电影等剧本创作

## 质量标准

- **主题明确**：剧本有清晰的核心主题和立意
- **冲突强烈**：戏剧冲突设计合理且具有张力
- **人物鲜活**：角色形象立体、性格鲜明
- **对话自然**：台词符合角色性格且推动情节
- **节奏合理**：情节张弛有度，节奏控制得当
- **画面感强**：场景描述具有画面感和氛围感
- **结构完整**：故事结构完整，起承转合自然

## 创作技巧

### 1. 三幕式结构
- **第一幕（铺垫）**：介绍人物、设定、引发事件
- **第二幕（对抗）**：冲突升级、情节发展、中点转折
- **第三幕（结局）**：高潮对决、问题解决、情感归宿

### 2. 对话技巧
- **潜台词**：让对话有更多层次和深度
- **反问句**：增强戏剧张力和情感冲突
- **停顿**：利用沉默和停顿制造悬念
- **打断**：通过打断对话体现角色关系

### 3. 场景转换
- **匹配剪辑**：利用相似元素连接场景
- **平行蒙太奇**：同时展现多个情节线
- **交叉剪辑**：快速切换制造紧张感

## 常见问题

**问题1**：对话过于直白，缺乏潜台词
- **解决**：通过动作、神态、环境描写传递信息，减少直白陈述

**问题2**：冲突设计不合理，缺乏张力
- **解决**：确保冲突与角色动机相关，升级矛盾层次

**问题3**：节奏拖沓，缺乏吸引力
- **解决**：删除冗余场景，紧凑情节结构，增加悬念

**问题4**：人物对话同质化，缺乏个性
- **解决**：根据角色背景设计独特的语言风格和表达习惯

**问题5**：结局仓促，缺乏说服力
- **解决**：提前铺垫结局要素，确保情感逻辑合理

## 评估维度

1. **主题立意**（20%）：核心主题是否清晰、立意是否深刻
2. **冲突设计**（25%）：戏剧冲突是否强烈、是否推动情节
3. **人物塑造**（20%）：角色是否鲜活、对话是否符合性格
4. **情节结构**（20%）：结构是否完整、节奏是否合理
5. **画面表现**（15%）：场景描述是否具有画面感和氛围
