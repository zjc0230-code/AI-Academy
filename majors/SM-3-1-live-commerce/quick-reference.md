# 直播带货快速参考

## 🎬 直播间搭建规范

### 硬件配置清单
```
基础设备（必需）：
1. 摄像设备
   - 手机：iPhone 12+ 或同等安卓手机
   - 相机：单反相机 + 采集卡（可选）
   - 要求：支持1080P/60fps

2. 音频设备
   - 麦克风：无线领夹麦克风或桌面麦克风
   - 要求：降噪好，音质清晰
   - 推荐：罗德、舒尔品牌

3. 网络设备
   - 网络：有线网络优先，5G/4G备用
   - 要求：上传速度≥10Mbps
   - 测试：开播前测速

4. 照明设备
   - 主灯：LED柔光灯（色温4500-5500K）
   - 补光灯：LED补光灯
   - 背景灯：背景装饰灯
   - 要求：光线均匀，无阴影

进阶设备（可选）：
1. 混音器：音频混音器
2. 推流设备：高清采集卡
3. 电脑：直播推流电脑
4. 显示器：监视器
```

### 直播间布局
```
标准直播间布局（3-5㎡）：

背景区（占比30%）：
- 背景：纯色背景或品牌背景
- 装饰：与产品相关的装饰
- 要求：整洁、专业、有品牌感

产品展示区（占比50%）：
- 展台：多层展示台
- 产品：核心产品突出展示
- 道具：产品相关道具
- 要求：产品清晰可见

主播区（占比20%）：
- 主播位置：黄金分割点
- 灯光：主光+补光+轮廓光
- 要求：主播形象清晰

设备区（角落）：
- 设备：推流设备、显示器
- 要求：不影响画面
```

### 环境要求
- **空间要求**：3-5㎡，安静无干扰
- **温度**：22-26℃，舒适环境
- **湿度**：40%-60%，防止设备受潮
- **光线**：明亮均匀，无强光直射
- **隔音**：良好隔音，避免回声

## 📝 直播流程策划

### 标准直播流程（60-90分钟）
```
开场环节（5-10分钟）：
1. 热场互动（2分钟）
   - 欢迎语："欢迎新进来的朋友"
   - 互动引导："来的朋友扣1"
   - 福利预告："今天有大福利"

2. 自我介绍（1分钟）
   - 主播介绍："我是XXX"
   - 直播主题："今天分享XXX"

3. 今日预告（2-3分钟）
   - 直播内容：今日直播内容
   - 产品预告：今日推荐产品
   - 福利预告：今日福利活动

4. 规则说明（1分钟）
   - 下单规则："怎么下单"
   - 互动规则："怎么参与互动"

产品展示环节（30-60分钟）：
每款产品5-10分钟：
1. 产品介绍（2-3分钟）
   - 产品展示：展示产品
   - 核心卖点：3个核心卖点
   - 使用场景：使用场景演示

2. 产品演示（2-3分钟）
   - 功能演示：演示产品功能
   - 效果展示：展示使用效果
   - 对比演示：与其他产品对比

3. 价格引导（1分钟）
   - 原价：产品原价
   - 直播价：直播优惠价
   - 优惠力度：优惠力度说明

4. 购买催单（1-2分钟）
   - 紧迫感："只有100件"
   - 限量："限量抢购"
   - 引导下单："赶紧去拍"

互动环节（穿插全程）：
1. 问答互动
   - 回答问题：回答用户问题
   - 互动引导：引导用户提问
   - 互动游戏：互动小游戏

2. 互动活动
   - 抽奖活动：直播间抽奖
   - 点赞活动：点赞送福利
   - 评论活动：评论有奖

收尾环节（5-10分钟）：
1. 今日总结（2分钟）
   - 产品回顾：今日推荐产品回顾
   - 数据回顾：今日数据回顾
   - 感谢致谢：感谢观众

2. 福利发放（2分钟）
   - 抽奖结果：公布抽奖结果
   - 福利发放：发放福利

3. 下次预告（1分钟）
   - 下次时间：下次直播时间
   - 下次内容：下次直播内容

4. 引导关注（1分钟）
   - 关注引导："关注主播"
   - 关注福利："关注有福利"
```

时间节点 | 内容 | 话术要点 | 备注

## 📦 商品选品策略

### 选品原则
```
选品三原则：
1. 目标受众匹配
   - 用户画像：目标用户画像
   - 需求匹配：满足用户需求
   - 价格匹配：符合用户消费能力

2. 平台匹配
   - 平台规则：符合平台规则
   - 平台调性：符合平台调性
   - 平台用户：符合平台用户

3. 自身匹配
   - 主播匹配：主播能讲解
   - 供应链：供应链稳定
   - 售后：售后有保障
```

### 选品标准
- **高需求**：市场需求大
- **高毛利**：毛利率≥30%
- **高复购**：复购率高
- **高品质**：产品质量好
- **高颜值**：产品颜值高
- **高口碑**：口碑评价好

### 选品流程
1. **市场调研**：
   - 热搜榜：查看平台热搜榜
   - 竞品分析：分析竞品选品
   - 用户调研：调研用户需求

2. **产品筛选**：
   - 初步筛选：根据标准筛选
   - 深度分析：分析产品数据
   - 测试推广：小范围测试

3. **供应链对接**：
   - 供应商选择：选择优质供应商
   - 价格谈判：谈判采购价格
   - 合同签订：签订采购合同

4. **产品测试**：
   - 样品测试：测试样品质量
   - 价格测试：测试价格接受度
   - 推广测试：测试推广效果
