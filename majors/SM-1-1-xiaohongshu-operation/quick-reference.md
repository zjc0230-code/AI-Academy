## 🎯 标题优化公式

### 公式1：数字+利益点
```
"3个步骤，让你的效率提升10倍"
"5个AI工具，帮你每天节省2小时"
"7个技巧，让你的笔记点击率翻倍"
```

### 公式2：痛点+解决方案
```
"工作太累？这个工具让你每天早下班"
"预算有限？这3个工具帮你免费搞定"
"时间不够？用这个方法1小时完成1天工作"
```

### 公式3：悬念+揭秘
```
"这5个AI工具，90%的人都还没用过"
"小红书爆款背后的秘密，只看这一篇就够了"
"我用了100个工具，最后只留下了这3个"
```

### 公式4：对比+反常识
```
"不是越贵越好！这3个免费工具比付费版更强"
"别再浪费时间了！这个工具1分钟搞定你1小时的工作"
"大多数人都错了！这才是正确的使用方法"
```

## 🎨 封面设计规范

### 构图法则
1. **黄金分割**：主要元素放在画面1/3处
2. **中心对称**：适合产品展示，视觉平衡
3. **对角线构图**：引导视线流动，增加动感

### 色彩要求
- **饱和度**：≥70%（鲜艳但不刺眼）
- **对比度**：明显（文字与背景对比）
- **配色方案**：
  - 红色系：活力、热情（适合美食、美妆）
  - 蓝色系：专业、科技（适合工具、职场）
  - 黄色系：温暖、亲和（适合生活、家居）

### 文字规范
- **字号**：主标题≥40px，副标题≥24px
- **字体**：粗体（Bold），无衬线字体
- **位置**：避开核心元素，留边距≥20px
- **数量**：不超过2行文字
