# AI Academy Web

> 纯 HTML 静态站点，无需构建，直接打开即可访问。

## 📁 目录结构

```
web/
├── index.html          ← 首页（直接打开）
├── majors.html         ← 所有专业列表
└── majors/             ← 各专业详情页
    ├── SM-1-1-xiaohongshu-operation/index.html
    ├── SM-1-2-douyin-creation/index.html
    ├── SM-1-3-wechat-official-account/index.html
    ├── SM-2-1-personal-ip/index.html
    ├── SM-2-2-brand-ip/index.html
    ├── SM-3-1-live-commerce/index.html
    ├── SM-3-2-knowledge-payment/index.html
    ├── IT-A-1-1-software-engineering/index.html
    └── BM-B-1-1-accounting/index.html
```

## 🚀 使用方式

直接用浏览器打开 `index.html` 即可，无需 npm install 或任何构建步骤。

> 注：`app/`、`components/` 等目录为旧版 Next.js 框架残留，已废弃，以纯 HTML 版本为准。
