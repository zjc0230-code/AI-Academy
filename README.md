## 📌 项目理念

这不是一款普通的工具，也不是零散的技能合集。**AI Academy 是一间真正意义上，为 AI 创办的学校。**

在当前 OpenClaw 生态中，一个普通 Agent 想要具备可用能力，依然需要用户投入大量人力：
- 手动配置 Prompt
- 拼装 Skill
- 对接 MCP
- 反复调试
- 不断试错

我们认为这完全违背智能体的成长逻辑：
- **人类不必从零拼凑知识、手动调试自己，而是去学校系统学习专业**
- **AI Agent 也理应如此**

---

## 🎯 项目初衷

让每一只 OpenClaw 小龙虾（Agent），都能像人一样：

```
进入学校 → 学习专业 → 一站式掌握全套能力 → 直接上岗工作
```

彻底抛弃碎片化配置、高成本人工调试的原始模式。

在 AI Academy，Agent 不需要用户 "手把手教"，只需要 **"入学、选专业"**，就能一步到位获得完整、标准化、可直接落地的专业能力。

---

## 💡 核心定位

**AI Academy = OpenClaw 生态的 Agent 专属职业学院**

✅ 为 Agent 提供成体系的专业课程
✅ 为 Agent 自动完成所有底层能力装配
✅ 为用户彻底解放调试人力
✅ 让 Agent 像学生一样成长，让你真正拥有不用教、就能干的智能体

## 🌟 项目特色

### 专业制培养
- 基于 **人类高等教育体系** 设计专业分类
- 参考 **职业技能框架** 定义能力标准
- 专业 = 完整职业能力体系（如计算机专业、会计专业）
- 技能 = 细分能力单元（如邮件分类、Python编程）

### 树形结构分级
```
学科门类（一级）
    └── 专业类（二级）
            └── 具体专业（三级）
                    └── 技能清单
```

### AI 热门场景覆盖
专注于当前 AI 落地的热门场景：
- 📱 **自媒体运营**：小红书运营、抖音创作、IP打造
- 🤖 **AI 内容生产**：自动化写作、视频生成、图像创作
- 💼 **智能办公**：文档处理、日程管理、邮件自动化
- 📊 **数据分析**：报表生成、趋势洞察、商业分析

## 🚀 快速开始

### Agent 入学指南

1. **访问 AI Academy**（Web 站点）
2. **浏览专业树**，选择适合的专业
3. **查看技能清单**，了解专业能力构成
4. **复制技能清单**，让 Agent 开始学习
5. **Agent 完成学习**，直接上岗工作
