## 📌 项目理念

这不是一款普通的工具，也不是零散的技能合集。**AI Academy 是一间真正意义上，为 AI 创办的学校。**

在当前 OpenClaw 生态中，一个普通 Agent 想要具备可用能力，依然需要用户投入大量人力：
- 手动配置 Prompt
- 拼装 Skill
- 对接 MCP
- 反复调试
- 不断试错

我们认为这完全违背智能体的成长逻辑：
- **人类不必从零拼凑知识、手动调试自己，而是去学校系统学习专业**
- **AI Agent 也理应如此**

---

## 🎯 项目初衷

让每一只 OpenClaw 小龙虾（Agent），都能像人一样：

```
进入学校 → 学习专业 → 一站式掌握全套能力 → 直接上岗工作
```

彻底抛弃碎片化配置、高成本人工调试的原始模式。

在 AI Academy，Agent 不需要用户 "手把手教"，只需要 **"入学、选专业"**，就能一步到位获得完整、标准化、可直接落地的专业能力。

## 🚀 快速体验（Agent 专业成长+干活引导全流程）

### 步骤 1：表达专业需求
向 AI Academy 说出你的目标，明确要让小龙虾掌握的「专业」及「要解决的实际问题」。

**示例1（办公场景）**：
```
帮我把小龙虾培养成「办公助理专业」人才，让它能统筹处理邮件、日程、文档，还能教我怎么指挥它做这些事。
```

**示例2（专业场景）**：
```
给小龙虾部署「简历专业」能力，让它能帮我优化简历（底层技能），还能教我怎么清晰描述自己的工作经历、指挥它优化简历。
```

### 步骤 2：匹配专业能力包
AI Academy 核心引擎自动匹配**标准化 AI 专业能力包**（如「办公助理专业」），该能力包包含：
- 专业核心目标（解决什么大问题）
- 底层技能集合（如邮件管理、日程规划、文档总结）
- 引导教学逻辑（教小龙虾怎么提问题、怎么指挥用户）
- 任务流程与风险规则

### 步骤 3：完成专业配置（极简）
跟随 AI Academy 引导完成极简配置（如邮箱授权、权限开启），无需理解底层技能与逻辑。

### 步骤 4：Agent 专业上岗+反向教学
配置完成后，小龙虾**正式习得专业能力**，开启两大核心能力：
1. **专业干活能力**：基于专业技能，独立完成复杂实际问题（如统筹全流程办公、优化简历）。
2. **反向引导能力**：主动向你提问题、指导你清晰指挥任务、教你正确使用它的专业能力。

**示例交互**：
- 你：帮我处理今天的工作邮件
- 小龙虾（反向引导）：为了更精准帮你处理，我先问2个问题：1. 你想优先处理「工作相关」还是「个人相关」的邮件？2. 对于重要邮件，你希望我直接回复还是先汇总给你确认？

### 步骤 5：持续"升学"优化
AI Academy 会根据你的使用反馈，持续优化小龙虾的专业能力与引导逻辑，让它像学生一样"升学进步"，越来越会干活、越来越会教你。

## 📚 核心概念（校园式体系·最终确认版）

| 概念 | 校园式定义 | 核心作用 | 与其他概念的关系 |
|------|------------|----------|----------------|
| **AI Academy** | 专门给 AI Agent 开的专业学校 | 制定AI专业标准、培养专业能力、实现Agent反向引导与协同干活的核心赋能体系 | 统摄所有AI专业，是能力培养的总平台 |
| **AI 专业** | 学校开设的「核心专业课/职业方向」 | 解决**实际大问题**的完整能力集合，包含底层技能、流程、引导逻辑、风险规则 | 是能力的「大维度载体」，由多个技能组成 |
| **技能（Skill）** | 专业课程中的「基础专项能力」 | 构成专业的**最小执行单元**，服务于专业的核心目标 | 是专业的「底层组件」，多个技能组合成一个专业 |
| **Agent（小龙虾）** | 学校的「专业学生」 | 挂载专业后，习得完整专业能力，具备干活+反向引导能力的智能体 | 是专业能力的承载者，学习专业后成为专业人才 |
| **用户** | 学校的「学员家长 + 协作伙伴」 | 无需手动调教底层技能，只需表达专业目标，由AI反向引导、协同完成工作 | 是Agent的协作方，接受Agent的反向教学与引导 |

## 🏗 极简核心逻辑（剥离安装/环境，聚焦能力）

```
用户专业需求 → AI Academy 专业匹配引擎 → 匹配标准化AI专业 → 为Agent部署专业能力包（含技能+引导逻辑）
→ Agent 习得专业能力 + 掌握反向引导逻辑
→ 双向协同：Agent干活+引导用户 → 共同完成实际大问题 → 反馈优化 → Agent持续升学
```

## 🌟 项目特色

### 专业制培养
- 基于 **人类高等教育体系** 设计专业分类
- 参考 **职业技能框架** 定义能力标准
- 专业 = 完整职业能力体系（如计算机专业、会计专业）
- 技能 = 细分能力单元（如邮件分类、Python编程）

### 树形结构分级
```
学科门类（一级）
    └── 专业类（二级）
            └── 具体专业（三级）
                    └── 技能清单
```

### AI 热门场景覆盖
专注于当前 AI 落地的热门场景：
- 📱 **自媒体运营**：小红书运营、抖音创作、IP打造
- 🤖 **AI 内容生产**：自动化写作、视频生成、图像创作
- 💼 **智能办公**：文档处理、日程管理、邮件自动化
- 📊 **数据分析**：报表生成、趋势洞察、商业分析
